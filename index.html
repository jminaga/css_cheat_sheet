Ever have trouble recalling the syntax for your favorite CSS code? Give it a permanent home and add it to this page!

Use these three properties to create a Flexbox row layout

.row {
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;
}

Use this to create a Flexbox column layout

.column { display: flex;
flex-direction: column;
}
